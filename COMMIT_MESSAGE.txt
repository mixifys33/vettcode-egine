Features:
- Free tier: 2 scans without login (localStorage tracking)
- Unlimited scans after authentication
- Login/Register with OTP email verification
- Users saved as "incomplete sellers" (no shop setup required)
- Persistent sessions with logout functionality
- Visual scan limit indicators

Technical:
- Created auth utilities (src/lib/auth.ts)

Backend Integration:
- POST /api/sellers/register - Register with OTP
- POST /api/sellers/verify - Verify OTP
- POST /api/sellers/login - Authenticate
- POST /api/sellers/resend-otp - Resend code

Security:

- OTP verification (6 digits, 10min expiry, 5 attempts)
- Account lockout protection
